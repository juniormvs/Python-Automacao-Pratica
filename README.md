# Python-Automacao-Pratica

Este repositório é dedicado a scripts práticos que demonstram o uso de Python para automatizar tarefas repetitivas e manipular dados, competências fundamentais para otimizar o fluxo de trabalho em Data Science e Machine Learning.

## Objetivos dos Projetos

* **Economia de Tempo:** Substituir tarefas manuais por scripts Python eficientes.
* **Pré-processamento de Dados:** Praticar a limpeza, extração e transformação de dados de diversas fontes (arquivos e web).
